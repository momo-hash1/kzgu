Поставка резервного источника электроснабжения КЗГУ АД-1200-Т400 (1200 кВт) на двигателе Yuchai для объекта OZON в Казани

21.02.2023

21.02.2023

Произведена поставка резервного источника электроснабжения КЗГУ АД-1200-Т400 (1200 кВт) на двигателе Yuchai для объекта OZON в Казани.

Дизельная электростанция на раме АД1200 открытая на базе двигателя YUCHAI. Предназначена для выработки электроэнергии резервного источника электропитания:

  • для населенных пунктов,

  • на стройплощадок,

  • для вахтовых поселков,

  • для установок сооружений для бурения,

  • в энергетических системах промышленных и производственных предприятий,

  • для учреждений образования,

  • для медицинских учреждений,

  • для обеспечения функционирования банков, гостиниц, торговых, складских комплексов и т.п.
